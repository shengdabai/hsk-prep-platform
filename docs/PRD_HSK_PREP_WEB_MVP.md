# PRD: HSK Prep Web MVP

## 定位

一个面向中文学习者的 HSK 对齐型备考网站 MVP，首发上线 HSK1，提供模考、专项练习、报告和错题本，同时具备后台审核与发布能力。

## 范围

### 学生端

- 首页
- HSK 级别页
- HSK1 模考列表
- HSK1 专项练习列表
- 做题页
- 提交评分
- 报告页
- 错题本
- 登录注册
- 价格页
- 账户页

### 管理端

- 管理后台首页
- 题目列表
- 题目详情 / 审核页
- 套卷管理
- 导入管理
- 发布管理
- 媒体管理
- 用户列表（只读）

## 角色

- anonymous：浏览公开内容和免费体验页
- learner：开始模考 / 专项，提交答案，查看报告和错题
- reviewer：admin 子集权限，负责审核
- admin：包含 reviewer 权限，并负责发布 / 下架和后台管理

## MVP 验收

1. 用户可登录、开始答题、提交评分并打开报告页。
2. 后台可浏览题目、审核状态、套卷与导入入口。
3. 服务端负责评分，前端不暴露正确答案。
4. 数据层有 SQL schema、seed、类型和导入脚本。
5. 项目可本地启动，并可接入 Supabase / Stripe。
